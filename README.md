# photo-editor
A photo editor with various (simple) effects, such as
- Horizontal & Vertical Flip
- Gray Scale
- Sepia Tone
- Colour Inversion
- Gaussian Blur
- Bulge Effect

Why we would need a bulge effect is anyone's guess. 8)
